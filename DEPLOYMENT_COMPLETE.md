## 🌐 **Acceso a la Aplicación**

### **Desarrollo Local:**
```bash
npm run dev
# → http://localhost:5173
```

### **Build de Producción:**

- `/upnassist/help` → Documentación legal
- `/matrix` → Pantalla Matrix (acceso manual)
